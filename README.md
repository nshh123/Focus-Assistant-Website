# Focus Assistant Website

This repository contains the landing page and promotional website for the **Focus Assistant** browser extension.

## Features

- **Modern Glassmorphism UI**: A sleek, dark-themed interface designed to promote focus and professionalism.
- **Dynamic Browser Detection**: The primary download CTA automatically adapts to the user's specific browser environment.
- **Fully Responsive**: Optimizations made to ensure seamless readability and presentation across mobile, tablet, and desktop devices.
- **Engaging Micro-Animations**: Smooth scroll effects and custom floating graphical elements.

## Architecture

This is a static website utilizing pure HTML, custom Vanilla CSS, and minimal Vanilla Javascript for interactions.
